
# AbstractionLayer

A collection of small libraries that provide interface layers for common .NET APIs so they can be mocked and unit-tested more easily.

This repository contains several packages targeting different areas of the platform:

- `DoenaSoft.AbstractionLayer.UI` - common UI abstractions shared by UI projects
- `DoenaSoft.AbstractionLayer.WinForms` - WinForms-specific abstractions
- `DoenaSoft.AbstractionLayer.WPF` - WPF-specific abstractions
- `DoenaSoft.AbstractionLayer.IO` - file I/O abstractions
- `DoenaSoft.AbstractionLayer.Web` - web / HTTP abstractions

Getting started

1. Clone the repository:

   `git clone https://github.com/DJDoena/AbstractionLayer.git`

2. Build with the .NET SDK (supports multi-targeting projects):

   `dotnet build`

Packages are available on NuGet (see individual project README files for package names and versions).
